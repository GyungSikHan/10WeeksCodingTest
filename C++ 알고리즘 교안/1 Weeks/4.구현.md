# 구현
- Tip)
    - 문제의 워딩 자체를 코드로 구현하는 문제
    - 문제를 하나하나 해결해 나가는것도 tip

----
1. 예시 문제) 아래 문자열에서 앞에서부터 3개의 문자열 출력
2. 예시 문제) 아래 문자열을 거꾸로 출력해라
3. 예시 문제) 해당 문자열 끝에 "umzunsik"이란 문자열을 추가해라

string dopa = "umzunsik";

```C++
#include<bits/stdc++.h>

using namespace std;

string dopa ="umzunsik";

int main()
{
    cout<<dopa<<"\n";

    //1)
    cout<<dopa.substr(0,3)<<"\n";
    //2)
    reverse(dopa.begin(), dopa.end());
    cout<<dopa<<"\n";
    //3)
    dopa += "umzunsik";
    cout<<dopa<<"\n";
}
```

- 예시 문제) : 숫자로 이뤄진 배열에 요소들의 오름차순 정렬 후 배열의 평균값 구하기
```C++
#include<bits/stdc++.h>

using namespace std;

int n;
int arr[104];
float aver;

int main()
{
    cin>>n;

    for(int i = 0; i < n; i++)
    {
        cin>>arr[i];
        aver += arr[i];
    }
    
    sort(arr, arr + n);

    for(int i = 0; i < n; i++)
    {
        cout<<arr[i]<<" ";
    }
    cout<<endl;

    aver/=n;
    cout<<fixed<<setprecision(2)<<aver;
}
```
