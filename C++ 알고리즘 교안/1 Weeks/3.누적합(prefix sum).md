# 누적합(prefix sum)
- 정의 
    - 요소들의 누적된 합의 의미
    - 어떤 배열 기반으로 앞의 요소들 부터 누적된 합을 새로이 저장하는 배열을 활용

- 누적합 예시
    - 기본 배열 arr[4] {1,10,11,100}
    - 누적합 배열 psum[4]
        - psum[0] = arr[0] : 1
        - psum[1] = psum[0]+arr[1] : 1 + 10 = 11
        - psum[2] = psum[1]+arr[2] :  1 + 10 + 11 = 22
        - psum[3] = psum[2]+arr[3] :  1 + 10 + 11 + 100 = 122

- 예시 문제) 특정 숫자 구간의 합을 구하라
```C++
#include <bits/stdc++.h>
using namespace std;

int n,m;
int temp;
int a,b;
int arr[100001];
int psum[100001];

int main()
{
    cin>>n>>m;
    for(int i = 1; i <= n; i++)
    {
        cin>>arr[i];
        psum[i] = arr[i] + psum[i-1]; 
    }
    
    for(int i =0;i<m;i++)
    {
        cin>>a>>b;

        cout<<psum[b] - psum[a-1]<<"\n";
    }
}
```
