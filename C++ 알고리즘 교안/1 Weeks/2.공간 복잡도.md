# 공간 복잡도
- 정의
    - 입력 크기에 대해 어떠한 알고리즘이 실행되는데 필요한 메모리 공간의 양
    - 요소들을 담을 공간이면 다 적용
- 알고리즘 문제풀 때 공간복잡도 활용
    1. 문제의 최대 범위
        - 문제에서 주어지는 최대 범위 이상의 컨테이너 잡기(최대 범위 + 1~4 정도면 적당)
    2. 메모리 제한
        - 문제에 나온 메모리 제한을 계산하여 그보다 작은 메모리로 잡으면 된다
        - 문제 풀때 매번 계산할 수 없음
        - Tip) 보통은 최저로 128MB로 주기 때문에 300만이라 보면 됨
            - 64MB : 1500만
            - 128MB :3000만
            - 256MB :6000만
            - 512MB :1억2000만
